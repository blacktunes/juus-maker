#### 使用方法

- 随便点

#### 说明

- 点击背景图和主图可修改图片
- 点击头像可打开头像选择
- 名字/内容可以直接点击修改
- 保存截图(可能存在兼容性问题)
  - 左侧区域的悬浮按钮可保存**JUUs**截图
  - 右侧区域的悬浮按钮可保存**完整对话**截图和**自动播放**
